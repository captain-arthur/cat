# Testkube 개요

## Testkube란?

Testkube는 쿠버네티스 클러스터에서 **다양한 테스트를 오케스트레이션하고 실행**하는 Kubernetes 네이티브 테스팅 플랫폼입니다.

| 속성 | 설명 |
|------|------|
| **목적** | Kubernetes에서 다양한 테스트 프레임워크를 자동화하여 실행 |
| **주체** | Kubeshop 개발 (Apache 2.0 라이선스) |
| **특징** | Kubernetes 네이티브, 다양한 테스트 프레임워크 지원, TestWorkflow |
| **언어** | Go |
| **지원 환경** | Kubernetes (네이티브 통합) |

## 주요 기능

| 기능 | 상세 설명 |
|------|----------|
| **다양한 테스트 프레임워크 지원** | Postman, k6, Cypress, Curl, SoapUI 등 다양한 테스트 도구 지원 |
| **TestWorkflow** | 여러 테스트를 단계별로 구성하여 복합 테스트 실행 |
| **Kubernetes 네이티브** | 테스트를 Kubernetes Job/Pod로 실행 |
| **CLI/API/Dashboard** | CLI, REST API, 웹 Dashboard 제공 |
| **트리거 및 스케줄링** | 수동, 스케줄, 이벤트, CI/CD 트리거 지원 |

## 주요 사용 사례

| 사용 사례 | 설명 |
|----------|------|
| **API 테스트 자동화** | Postman, Curl 등으로 API 테스트 자동화 |
| **성능 테스트** | k6, JMeter 등으로 성능 테스트 실행 |
| **E2E 테스트** | Cypress, Playwright 등으로 E2E 테스트 실행 |
| **복합 테스트 워크플로우** | 여러 테스트를 순차/병렬로 구성하여 실행 |
| **CI/CD 통합** | CI/CD 파이프라인에서 테스트 자동화 |

## 장점

| 장점 | 설명 |
|------|------|
| **Kubernetes 네이티브** | 테스트를 Kubernetes 리소스로 실행하여 관리 용이 |
| **다양한 프레임워크 지원** | Postman, k6, Cypress 등 다양한 테스트 도구 지원 |
| **TestWorkflow** | 복잡한 테스트 시나리오를 Workflow로 구성 가능 |
| **CLI/API/Dashboard** | 다양한 인터페이스 제공으로 편의성 향상 |
| **트리거 및 스케줄링** | 다양한 트리거 방식으로 자동화 용이 |

## 제약 사항

| 제약 | 설명 |
|------|------|
| **Kubernetes 전용** | Kubernetes 환경에서만 실행 가능 |
| **리소스 요구** | 테스트 실행을 위한 충분한 리소스 필요 |
| **학습 곡선** | 다양한 프레임워크별 설정 방법 학습 필요 |

## Sonobuoy, ClusterLoader2, kube-burner, Chaos Toolkit, Chaos Mesh, LitmusChaos, Kubescape, kube-bench와의 관계

| 도구 | 목적 | 특징 |
|------|------|------|
| **Sonobuoy** | Conformance 테스트 (기본 자격 검증) | 기능 정상 동작 여부 검증 |
| **ClusterLoader2** | 성능/스케일 테스트 | 내장 measurement, 구조화된 리포팅 |
| **kube-burner** | 성능/스케일 테스트 | Prometheus 중심 관측/리포팅 |
| **Chaos Toolkit** | 복원력/장애 대응 테스트 | Python 기반, Steady-State Hypothesis |
| **Chaos Mesh** | 복원력/장애 대응 테스트 | Kubernetes 네이티브, CRD 기반, Dashboard |
| **LitmusChaos** | 복원력/장애 대응 테스트 | Kubernetes 네이티브, ChaosHub, Probes |
| **Kubescape** | 보안 기본선 검증 | 보안 스캔, 취약점 탐지, 프레임워크 기반 |
| **kube-bench** | 보안 기본선 검증 | CIS Benchmark 특화, 구성 설정 감사 |
| **Testkube** | 테스트 자동화 | 다양한 테스트 프레임워크 지원, TestWorkflow |

## CAT 관점에서의 역할

Testkube는 **쿠버네티스 클러스터 인수테스트(CAT)**의 **"테스트 자동화(Test Automation)"** 카테고리에 해당하는 핵심 도구입니다.

### 검증 질문(Assertion) 예시

- "다양한 테스트를 Kubernetes에서 자동화하여 실행할 수 있는가?"
- "API 테스트, 성능 테스트, E2E 테스트를 자동화할 수 있는가?"
- "복합 테스트 워크플로우를 구성하고 실행할 수 있는가?"

자세한 내용은 [06-CAT-활용.md](./06-CAT-활용.md) 참조.
