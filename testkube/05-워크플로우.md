# Workflow

Testkube TestWorkflow를 사용하여 여러 테스트를 단계별로 구성하는 방법을 설명합니다.

## TestWorkflow란?

TestWorkflow는 여러 Test를 단계별로 구성하여 복합 테스트를 실행하는 기능입니다.

### 주요 특징

- **순차 실행**: 테스트를 순차적으로 실행
- **병렬 실행**: 여러 테스트를 동시에 실행
- **매트릭스 실행**: 다양한 조합으로 테스트 실행

## TestWorkflow 기본 구조

```yaml
apiVersion: tests.testkube.io/v1
kind: TestWorkflow
metadata:
  name: my-workflow
spec:
  steps:
    - name: api-test
      test: my-api-test
    - name: perf-test
      test: my-perf-test
```
