# 실무 체크리스트

Kubescape를 실무에 적용할 때 확인해야 할 사항들을 정리합니다.

## 실행 전 체크리스트

### 1. 환경 준비

| 항목 | 확인 사항 | 확인 방법 |
|------|----------|----------|
| **Kubernetes 버전** | Kubernetes 1.17 이상 지원 여부 확인 | `kubectl version --short` |
| **Kubescape 설치** | Kubescape 설치 여부 확인 | `kubescape version` |
| **kubeconfig** | Kubernetes 클러스터 연결 확인 | `kubectl cluster-info` |
| **권한** | Kubernetes 리소스 읽기 권한 확인 | `kubectl auth can-i get pods --all-namespaces` |

### 2. 스캔 대상 준비

| 항목 | 확인 사항 | 비고 |
|------|----------|------|
| **스캔 대상** | 클러스터 또는 Manifest 파일 확인 | 클러스터/파일 스캔 선택 |
| **프레임워크 선택** | 사용할 프레임워크 확인 (NSA, CIS, MITRE 등) | 조직 기준에 맞게 선택 |
| **합격 기준** | Compliance threshold 설정 | 기본값 80% |

## 실행 중 체크리스트

### 1. 스캔 실행

| 항목 | 확인 방법 | 비고 |
|------|----------|------|
| **스캔 상태** | 스캔 진행 상황 확인 | 터미널 출력 확인 |
| **에러 확인** | 스캔 중 에러 발생 여부 확인 | 로그 확인 |

## 실행 후 체크리스트

### 1. 결과 확인

| 항목 | 확인 방법 | 비고 |
|------|----------|------|
| **점수 확인** | 스캔 점수 확인 | JSON 리포트 내 `summary.score` |
| **합격 여부** | Compliance threshold 대비 점수 확인 | 합격 기준 충족 여부 |
| **취약점 확인** | Critical/High 취약점 확인 | 이미지 스캔 결과 확인 |

### 2. 리포트 생성

| 산출물 | 생성 방법 | 비고 |
|--------|----------|------|
| **JSON 리포트** | `kubescape scan framework cis --format json --output results.json` | 원본 리포트 |
| **JUnit XML** | `kubescape scan framework cis --format junit --output results.xml` | CI/CD 통합용 |
| **HTML 리포트** | `kubescape scan framework cis --format html --output report.html` | 시각화 리포트 |
