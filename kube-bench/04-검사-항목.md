# 검사 항목

kube-bench가 검사하는 CIS Kubernetes Benchmark 항목들을 상세히 설명합니다.

## 검사 대상

### 1. 마스터 노드 (Control Plane)

| 검사 항목 | 설명 |
|----------|------|
| **API Server** | API Server 설정 검사 |
| **Scheduler** | Scheduler 설정 검사 |
| **Controller Manager** | Controller Manager 설정 검사 |
| **etcd** | etcd 설정 검사 |
| **Network** | 네트워크 설정 검사 |
| **Authentication & Authorization** | 인증/인가 설정 검사 |

### 2. 워커 노드 (Worker Node)

| 검사 항목 | 설명 |
|----------|------|
| **kubelet** | kubelet 설정 검사 |
| **kube-proxy** | kube-proxy 설정 검사 |
| **Network** | 네트워크 설정 검사 |

## 검사 결과

### 결과 상태

| 상태 | 설명 |
|------|------|
| **PASS** | 검사 항목 통과 |
| **FAIL** | 검사 항목 실패 |
| **WARN** | 경고 항목 |
| **INFO** | 정보 제공 항목 |

### 예시 출력

```
[PASS] 1.1.1 Ensure that the API server pod specification file permissions are set to 644 or more restrictive
[FAIL] 1.1.2 Ensure that the API server pod specification file ownership is set to root:root
[WARN] 1.1.3 Ensure that the controller manager pod specification file has permissions set to 644 or more restrictive
[INFO] 1.1.4 Ensure that the scheduler pod specification file has permissions set to 644 or more restrictive
```
