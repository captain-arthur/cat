# Workflow

LitmusChaos Workflow를 사용하여 여러 실험을 직렬/병렬로 구성하는 방법을 설명합니다.

## ChaosWorkflow란?

ChaosWorkflow는 여러 Chaos 실험을 단계별로 구성하여 복합 실험을 실행하는 기능입니다.

### 주요 특징

- **직렬 실행**: 실험을 순차적으로 실행
- **병렬 실행**: 여러 실험을 동시에 실행
- **Probes 통합**: 각 단계에서 Probe 검증

## Workflow 기본 구조

```yaml
apiVersion: litmuschaos.io/v1alpha1
kind: ChaosWorkflow
metadata:
  name: workflow-example
  namespace: default
spec:
  workflow_id: workflow-example
  workflow_manifest: |
    # Workflow 정의 (YAML)
```
